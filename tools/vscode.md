# VSCode 개발 환경

## VSCode Workspace
여러 프로젝트를 한 VSCode에서 동시에 관리하는 기능

### 설정 방법
1. File → Add Folder to Workspace
2. 관련된 모든 프로젝트 폴더 추가:
   - TIL
   - Portfolio
   - (나중) React 프로젝트
3. File → Save Workspace As → my-workspace.code-workspace

### 장점
- 한 VSCode 창에서 여러 프로젝트 관리
- 터미널 여러 개 동시 실행
- 파일 간 쉬운 비교

---

## Split View (화면 분할)
한 VSCode 화면을 좌우로 나눠서 파일 비교

### 단축키
- **Ctrl + \** : 화면 분할
- **Ctrl + Tab** : 파일 전환

### 사용 예시
- 좌측: 게임 코드 (script.js)
- 우측: 포트폴리오 코드 (style.css)
