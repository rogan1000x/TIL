# Day 09 - 최고 점수 저장 + 게임 카운터

## 오늘 배운 것
- **localStorage**: 브라우저에 데이터 저장 (새로고침/페이지 종료해도 사라지지 않음)
- **localStorage.getItem()**: 저장된 데이터 불러오기
- **localStorage.setItem()**: 데이터 저장하기
- **offsetHeight**: 강제 리플로우 (DOM 재계산) 후 애니메이션 초기화
- **classList.remove() + add()**: 클래스 제거 후 다시 추가해서 애니메이션 재실행

## 게임에 적용한 것
- 최고 점수를 localStorage에 저장
- 게임 시작할 때마다 저장된 최고 점수 불러오기
- 최고 점수보다 높은 점수 나오면 자동 업데이트
- 스타트 페이지와 게임오버 화면에 최고 점수 표시
- 게임 시작 시 3, 2, 1 카운터 추가
- 카운터가 번쩍거리면서 변경되는 애니메이션
- 카운터 후 게임 자동 시작

## 막혔던 것
- showCountdown 함수가 중복 정의돼서 게임이 안 됨
  → 전체 코드를 정리해서 해결
- 카운터가 팟팟팟 변경되던 것
  → offsetHeight로 강제 리플로우 후 애니메이션 재실행
