# Day 01 - HTML/CSS 기초

## 오늘 배운 것
- HTML은 뼈대: 화면에 뭘 보여줄지 구조를 잡는 것
- CSS는 옷: 색깔, 크기, 위치 등 스타일을 입히는 것
- `<div>` 는 그냥 박스, 용도에 따라 id로 이름 붙임
- `display: flex` 로 요소들 가로/세로 정렬 가능
- `position: relative` 는 자식 요소들의 기준점이 됨

## 게임에 적용한 것
- 검은 배경 (#050a0f) + 네온 초록 (#00ffcc) 사이버펑크 화면 완성
- HUD (점수판) 상단에 배치
- 게임 영역 가운데, 입력창 하단에 배치
- Live Server로 저장하면 브라우저가 자동으로 새로고침되는 것 확인
