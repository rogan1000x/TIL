# Day 10 - 모바일 반응형 + 사운드 효과 완성

## 오늘 배운 것
- **@media 쿼리**: CSS 반응형 디자인 (화면 크기별로 다른 스타일)
- **Web Audio API**: 외부 파일 없이 코드로 소리 생성
- **AudioContext**: 사운드 생성 및 제어
- **Oscillator**: 진동수로 소리 음정 조절
- **GainNode**: 소리 크기 조절 (volume)

## 게임에 적용한 것
- 모바일 반응형 추가:
  - 768px 이하: 태블릿 크기 조정
  - 480px 이하: 작은 폰 크기 조정
  - 375px 이하: 초소형 폰 크기 조정
  
- 사운드 효과 5가지:
  - 시작 버튼음: 500Hz → 700Hz (준비 완료)
  - 카운트음: 900Hz (3, 2, 1 각 숫자마다)
  - 성공음: 800Hz (단어 맞췄을 때)
  - 실패음: 400Hz (단어 놓쳤을 때)
  - 게임오버음: 600Hz + 400Hz (경고음)
  - 재시작음: 600Hz (버튼 클릭)
