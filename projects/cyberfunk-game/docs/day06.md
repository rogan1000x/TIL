# Day 06 - 게임오버 화면 구현

## 오늘 배운 것
- `display: none` → 요소 숨기기
- `display: flex` → 요소 보이게 하기
- `position: absolute` → 부모 요소 기준으로 위치 잡기
- `window.onload` → 페이지 완전히 로드된 후 실행
- `Math.max(0, n)` → 0 아래로 안 내려가게 막기
- 콘솔 에러 읽는 법 (F12 → Console 탭)

## 게임에 적용한 것
- HP 0 되면 게임오버 화면 표시
- 최종 점수 게임오버 화면에 표시
- 재시작 버튼으로 게임 리셋

## 막혔던 것
- `isGameOver` 중복 선언 에러
  → script.js 전체 지우고 다시 붙여넣어서 해결
- 재시작 버튼 null 에러
  → window.onload 안에 넣어서 해결
- 화면이 2개로 나뉘어 보임
  → 브라우저 창이 너무 넓어서 생기는 문제, 나중에 수정 예정
