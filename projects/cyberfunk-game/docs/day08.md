# Day 08 - 콤보 시스템 + 게임오버 화면 + 애니메이션

## 오늘 배운 것
- **콤보 시스템**: 연속으로 단어를 맞추면 점수가 배수로 올라감
- **classList.add/remove**: CSS 클래스를 JavaScript에서 동적으로 추가/제거
- **setTimeout**: 애니메이션 후 자동으로 클래스 제거해서 다시 쓸 수 있게 함
- **@keyframes**: CSS 애니메이션 정의
- **z-index**: 요소의 레이어 순서 조절
- **position: fixed vs absolute**: fixed는 화면 전체 고정, absolute는 부모 기준

## 게임에 적용한 것
- 콤보 시스템: 단어 맞출 때마다 콤보 +1, 놓칠 때 0으로 리셋
- 점수 계산: 10 × 콤보 (콤보 1: +10점, 콤보 2: +20점...)
- 점등 애니메이션: 점수, 콤보, 레벨, HP 변할 때 번쩍거리는 효과
- 게임 테두리 점등: 단어 맞추면 초록 테두리 점등, 놓치면 빨간색 흔들림
- 게임오버 화면: position: fixed로 전체 화면을 완전히 덮음
- 게임 스타트 페이지: 시작 버튼으로 게임 시작

## 막혔던 것
- script.js에 중복 코드가 있어서 HP 깎일 때 렉이 생김
  → 전체 코드를 정리해서 해결
- index.html이 두 번 들어가있어서 중복 선언 에러
  → 중복된 부분 전부 삭제
- 게임오버 화면에 떨어지는 단어가 보임
  → position: fixed + z-index: 9999 + 더 불투명한 배경으로 해결
